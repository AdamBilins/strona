Dział 1

Naucz się podstaw frontendu. Dowiedz się co to HTML i poznaj jego strukturę.
Znajdziesz tu podstawowe znaczniki.

Co to HTML?

HTML to Język znaczników, służący do budowy struktury (szkieletu strony).

Technologie frontendowe

HTML + CSS + JS + „RWD”

Podstawowa struktura strony

<!DOCTYPE html>
<html lang="pl">
<head>
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial- scale=1.0" />
<title>Document</title>
</head>
<body></body>
</html>

!DOCTYPE html - Deklaracji dokumentu HTML, informuje przeglądarkę o rodzaju dokumentu, który ma przetworzyć
html lang="pl" - Lang deklaruje język używany na stronie
Head - - sekcja nagłówkowa w dokumencie HTML, znajduje się tu m.in metainformacje, tytuł karty
Html - Definicja dokumentu
Body - Treść strony, wszystko co widoczne

RWD

Responsywne projektowanie stron internetowych (ang. responsive web design, RWD) – technika projektowania strony WWW, aby jej układ dostosowywał się samoczynnie do rozmiaru okna przeglądarki, na której jest wyświetlany np. przeglądarki, smartfonów czy tabletów.

meta charset="UTF-8" - kodowanie polskich znaków, deklaracja zestawu znaków, który będzie używany na stronie internetowej, aby tekst wyświetlał się odpowiednio
meta - metainformacje to dodatkowe (w większości opcjonalne) wiadomości pozwalające na umieszczenie w dokumencie informacji np. kodowanie znaków na stronie internetowej. Meta znajduję się w sekcji head
title - - m.in. do nadania tytuły karty
Nagłówki h - nagłówki na stronie, od h1 do h6. Cyfry od 1 do 6 oznaczają hierarchię nagłówków, zaczynając od 1 jako najważniejszego w hierarchii. h1 może być użyty na stornie tylko raz, inne mogą wiele razy się powtarzać. Nagłówki pozwalają użytkownikom i robotom Google, lepiej odnaleźć się w treściach na stronie. Wyszukiwarka indeksuje witrynę między innymi zgodnie z tym, co umieszczone w nagłówkach oraz w jakiej kolejności oraz ilości one wystąpią.
p - paragraf, akapit
b - pogrubienie (niesemantyczne)
i - kursywa (niesemantyczne)
u, ins - podkreślenie (niesemantyczne)
del - przekreślenie
s - przekreślenie, gdy chcemy skreślić nieaktualną treść na stronie, a nie chcemy jej usuwać
strong - pogrubienie (semantyczne, czyli gdy chcemy coś wyróżnić)
em - podkreślenie (semantyczne, czyli gdy chcemy coś wyróżnić)
sup - index górny
sub - index dolny
small - znaczne zmniejsza tekst
q - Cytat: („tekst”)
blockquote - Cytat (z wcięciem) (odstęp tekst)
cite - Cytat (z pochyleniem), używany gdy chcemy odwołać się do jakiejś publikacji, pracy lub utworu literackiego
br - łamanie linii, przejście do nowego wiersza
abbr title="profesor" | prof. | /abbr - skrótowce, czyli skróty (najechanie pokazuje rozwinięcie skrótu). Skrót jest domyślnie podkreślony kropkami. Abbr używamy do skrótów, które nie mają kropki.
acronym - służy do oznaczania akronimów, czyli bez kropki (najechanie pokazuje rozwinięcie skrótu), Skrót jest domyślnie podkreślony kropkami
dfn - używamy do wyjaśnienia jakiegoś terminu lub pojęcia
center - Do wyśrodkowania treści na stronie, znacznik przestarzały (nieużywany, do wyśrodkowania służy css)
font face | size | color - Zmiana czcionki, kroju czcionki | Zmiana rozmiaru czcionki | Zmiana koloru czcionki
Font size zapisujemy w cyfrach od 1 od 7 (nie w px itd.)

Znaczniki Semantyczne - Nowości HTML 5 m.in:
footer, nav, section, header, main

Znaczniki Zdeprecjonowane m.in:
frame, frameset, noframes, font, center

Znaczniki Semantyczne i Zdeprecjonowane

Znaczniki Semantyczne to takie znaczniki którego nazwa opisuje jego zawartość i nie posiada szczególnych atrybutów. Elementy semantyczne umożliwiają intuicyjny podział dokumentu i mogą być szczególnie pomocne dla wyszukiwarek.

Znaczniki Zdeprecjonowane to przestarzałe znaczniki lub atrybuty, które zostały uznane za zbędne i zastąpione innym, równoważnym rozwiązaniem np. w css.

Semantyka HTML: Semantyka HTML to najprościej mówiąc wykorzystanie znaczników semantycznych, zgodnie z tym do czego zostały one stworzone. Ponieważ semantyka html jest jedną z podstawowych zasad dobrego SEO (SEO czyli Optymalizacja dla wyszukiwarek internetowych).

Semantyka ma również duży wpływ na przejrzystość kodu.
Wykorzystywanie semantycznych znaczników HTML może przyspieszyć indeksowanie strony przez roboty wyszukiwarek.
Semantyczne znaczniki HTML pomagają w lepszym zrozumieniu kodu strony

Framework

Framework: Framework jest narzędziem, które dostarcza gotowe komponenty lub rozwiązania. Te są dostosowywane w celu przyspieszenia rozwoju danego projektu. Framework może zawierać bibliotekę lub biblioteki, do których możemy sięgać w trakcie pracy. Dodatkowo framework może obejmować programy pomocnicze, kompilatory, biblioteki kodu, zestawy narzędzi. Dzięki frameworkom tworzenie staje się bardziej efektywne.

Framewokr-ki:
Bootstrap to popularny framework CSS
jQuery to biblioteka JavaScript
TypeScript to język programowania stworzony przez firmę Microsoft jako nadzbiór języka JavaScript
React jest biblioteką języka programowania JavaScript
React Native służy do tworzenie aplikacji wieloplatformowych JavaScript
Angular to framework i platforma do tworzenia SPA, napisany w języku TypeScript
Vue do aplikacji webowych typu front-end, oparty na języku JavaScript oraz modelu Model-View-Controller
NativeScript do tworzenia aplikacji mobilnych na platformy Apple iOS i Android
