HTML czyli HyperText Markup Language (pol. hipertekstowy język znaczników)


Jest to język służący do tworzenia dokumentów hipertekstowych oraz opisywania struktury informacji zawartych wewnątrz strony. Aktualnie obowiązującą wersją HTML jest W3C HTML 5. HTML opisuje strukturę stron internetowych, a CSS określa formę jej prezentacji w przeglądarce.

Co możemy zrobić w HTML?


O standardy pisania stron internetowych (a co za tym idzie o proces normalizacji HTML) dba organizacja World Wide Web Consortium (W3C). Za pomocą języka HTML nadajemy określone znaczenie semantyczne na stronie dla elementów treści takich jak: nagłówki (H1 - H6) akapity tekstu hiperłącza (linki) formularze listy i inne HTML umożliwia także dostosowanie wyglądu treści dokumentu w przeglądarce internetowej (np. Google Chrome) przy wykorzystaniu na stronie CSS (ang. Cascading Style Sheets) czyli kaskadowych arkuszy stylów.

Web Developer zajmuje się tworzeniem efektywnego, testowalnego kodu, budowaniem witryny internetowej od projektu po wykończenie, dba o design witryny.
Stanowisko web developera jest odpowiedzialne za wszystko to, co widzisz, otwierając strony internetowe. To właśnie web developer zajmuje się tworzeniem witryn internetowych i aplikacji webowych. Czy robi to wszystko sam? Nie, pracuje w zespole, w którym każdy programista zajmuje się inną częścią projektów. Jest to zawód przyszłości. Internet stał się niczym żywy organizm i ewoluuje w zastraszającym tempie. Z tego powodu na rynku pracy możemy zaobserwować ogromne zapotrzebowanie na web developerów. Jeśli interesujesz się nowymi technologiami oraz programowaniem, warto zastanowić się, czy przypadkiem praca web developera nie jest idealną opcją.


Web development to po prostu tworzenie witryn internetowych. Jest to dziedzina, która polega na tworzeniu oprogramowania, dzięki któremu korzystając ze stron internetowych, możemy korzystać w wielu funkcji takich jak, chociażby interfejs użytkownika. Wszystkie firmy działające w sieci lub posiadające swoje strony muszą podjąć współpracę z działem IT, jakim jest web development. Projekty web developmentu realizuje… web developer. To właśnie on jest jednym z elementów składowych, przekładających się na sprawne funkcjonowanie stron internetowych.

Web developer to osoba, która zna tajniki programowania. Tworzy kod, który zapewnia sprawne działanie strony lub aplikacji. Dopasowuje on język programowania i technologię do wymagań strony. Dodatkowo jego praca polega także na tworzeniu dokumentacji technicznej przeznaczonej dla użytkownika oraz wychwytuje i diagnozuje błędy pojawiające się podczas użytkowania strony. Jest on także częścią zespołu. Zazwyczaj współpracuje z web desingerem, który tworzy stronę graficzną projektów. Co ciekawe, swoje umiejętności web developer musi też wykorzystać w oparciu o współpracę z copywriterem, który jest odpowiedzialny za całą warstwę tekstową danej strony bądź aplikacji. Możliwość rozwoju to jedna z kluczowych zalet tego zawodu. Wraz ze wzrostem doświadczenia i umiejętności możesz liczyć na ciekawą ścieżkę kariery. Co więcej, web developer ma kilka specjalizacji do wyboru. Najpopularniejsze z nich to: Front end developer — zajmuje się wszystkim tym, co widzimy na stronie. Odpowiada on za jej wygląd i mechanikę. Back end developer — funkcjonalność techniczna, płynność działania, bezpieczeństwo. To, co niewidoczne dla użytkownika, a kluczowe dla działania strony, tworzy właśnie back end.
